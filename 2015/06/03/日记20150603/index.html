    以上问题明天继续。

命令：
wget [参数] [URL地址]
wget默认下载到当前目录
[参数] -b 后台执行，tail -f wget-log查看进度 -c 继续下载 -P 指定目录 -O 保存文件名
wget —ftp-user=USERNAME —ftp-password=PASSWORD url 下载ftp

bzip2 [参数 文件]
-d —decompress 解压，默认解压
-z —compress 压缩，可以使用命令bunzip2
-c —stdout 输出到标准输出，和-d一起等同于使用命令bzcat

patch [OPTION]… [ORIGFILE [PATCHFILE]]
-p 指定目录级别 p0完整路径 p1去除斜杠 p2去除斜杠和第一个目录
-i 从指定文件读取补丁
-R 逆向补丁
patch-p0 < xxx.patch

diff [OPTION]… FILES
diff -Naru xxx1 xxx2 > xxx.patch
