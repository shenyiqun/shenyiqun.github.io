Ubuntu12.04.5升级3.18.14

下载：wget www.kernel.org/pub/linux/kernel/v3.x/linux-3.18.14.tar.xz
复制到/usr/src: sudo cp linux-3.18.14.tar.xz /usr/src
解压：sudo tar vxf linux-3.18.14.tar.xz
进入linux-3.18.14目录，复制默认配置：
cd linux-3.18.14
sudo cp /boot/config* .config
sudo make menuconfig
sudo make
sudo make modules
sudo make modules_install
sudo mkdir /boot/config-3.18.14
sudo mkinitramfs -o initrd.img-3.18.14 3.18.14
sudo cp arch/x86/boot/bzImage /boot/vmlinuz-3.18.14
sudo cp initrd.img-3.18.14 /boot
到此升级完成，接下来配置grup
grub-mkconfig
update-grub更新配置文件
重启长按shift可以进入grub菜单。
